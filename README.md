# Transformer_Safety_Device
A hardware-based automatic protection system that safeguards electrical transformers from damage caused by overloading — without any microcontroller or software programming.

230V AC Mains
      ↓
Step-Down Transformer (230V → 12V AC)
      ↓
Rectifier & Filter (AC → Smooth DC)
      ↓
Current Sensor monitors load current
      ↓
Timer Circuit confirms overload
      ↓
Relay triggers → Load disconnected
      ↓
Transformer Protected ✅

Key Features
No microcontroller - Pure hardware design
Automatic protection - No manual intervention needed
4 load circuits- L1, L2, L3, L4 independently controlled
Backup power-9V battery for control circuits
Timer delay-Prevents nuisance tripping
Auto reset-Resets after fault is cleared
Cost effective-Simple and affordable components
